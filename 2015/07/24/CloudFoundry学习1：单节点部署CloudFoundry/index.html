Cloud Foundry

Cloud Foundry是VMware于2011年4月12日推出的业界第一个开源PaaS云平台，它支持多种框架、语言、运行时环境、云平台及应用服务，使开发 人员能够在几秒钟内进行应用程序的部署和扩展，无需担心任何基础架构的问题。

环境要求

以Ubuntu 10.04 Server 64-bit为例，记住一定要是64位的，因为CF是构建在64位架构上。一个Ubuntu环境就够了，接下来就是按照文档来单节点部署一个实例，事实上在生产环境就是应该多节点安装，每个模块可以分别安装在不同的VM上面，但是开发和实验环境为了方便，就安装在一台VM上就足够了。
