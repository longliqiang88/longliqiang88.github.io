注解

相当于一种标记，在程序中加入注解就是为程序打上了某种标记。加入标记后，Java编译器可以利用反射机制来了解类和各种元素上有无任何标记，然后根据标记去做相应的事情。标记可以加在包、类、字段、方法、方法的参数以及局部变量上。
